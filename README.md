# \# Structured-light-extraction-method

# 

# This repository provides the implementation of our structured light extraction method. It focuses on the robust and high-precision extraction of laser stripe centers from structured light patterns, which is a critical foundational step for subsequent 3D reconstruction and calibration tasks.
